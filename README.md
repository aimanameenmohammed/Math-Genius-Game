A high-performance, interactive educational engine developed in **C#** and **WinForms**. This project transcends the traditional "Math Game" logic by implementing modern UI/UX principles, efficient event handling, and a modular architecture.

---

## 🏗️ Technical Architecture & Design Patterns

This project was built with a focus on **Clean Code** and **Modular Engineering**:

* **⚡ Unified Event Handling:** Implemented a centralized Event Handler for multiple UI components to reduce memory footprint and ensure "Clean Code" standards.
* **📐 Custom UI Engineering:** Leveraged GDI+ concepts to implement rounded corners (Round Borders) and modern aesthetics, breaking the standard "Boxy" WinForms look.
* **🧩 Modular Functionality:** Engineered a dedicated Static Class to handle UI state transitions (RadioButtons, Buttons, and Panels) providing seamless flexibility.
* **⚙️ Game Logic Optimization:** Migrated from C++ Console logic to a dynamic GUI-based state machine, ensuring high responsiveness and zero-lag user interaction.
* **🛡️ Data Integrity:** Used customized Radio-Logic to ensure precise user selection during high-speed gameplay.

---

## 🎮 Key Features

* **Dynamic Difficulty Scaling:** Choose between various difficulty levels and operation types.
* **Time-Trial Mode:** Integrated real-time countdown to challenge the user's cognitive speed.
* **In-Place Dashboard:** Results are displayed via a dynamic Panel overlay system, eliminating the need for redundant pop-up windows.
* **Immersive Experience:** High-quality audio integration for an engaging gaming session.
